# Helm Deployment

This directory will contain the production Kubernetes chart for Insight Notebook.

Planned workloads:

- `frontend`
- `api`
- `parser-worker`
- `embedding-worker`
- `model-gateway`
- `scheduler`

Planned dependencies:

- PostgreSQL
- OpenSearch
- S3-compatible object storage
- OIDC provider configuration
- OpenTelemetry collector
